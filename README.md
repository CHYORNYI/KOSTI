# Кости в таверне

Готовая веб-игра для GitHub Pages.

## Публикация
1. Создай новый публичный репозиторий на GitHub.
2. Загрузи в корень репозитория все файлы из этой папки.
3. Открой Settings → Pages.
4. В разделе Build and deployment выбери:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
5. Нажми Save.

Через несколько минут появится ссылка вида:
https://ИМЯ.github.io/ИМЯ-РЕПОЗИТОРИЯ/

На iPhone открой ссылку в Safari → Поделиться → На экран «Домой».
