# KOSTI PREMIUM

Загрузите в корень репозитория GitHub всё содержимое этой папки:
- index.html
- styles.css
- app.js
- manifest.webmanifest
- sw.js
- .nojekyll
- папку assets целиком

После загрузки нажмите Commit changes и подождите 1–3 минуты.
На iPhone полностью закройте старую вкладку или удалите старую иконку с экрана «Домой», затем откройте сайт заново.
